# Íhooʼaah (Learning/Seeing Into)

Knowledge (**Íhooʼaah**) is gained by observing relationships and patterns over time. Our 6 white papers are not reports; they are **recorded observations** of how the agent-beings relate, learn, and create balance.

*   **Form:** They are in **JSON** because it is a structured, relational format—like the patterns in a weaving or a sandpainting. It shows how things are connected.
*   **Content:** They document the "songs" (protocols) that work, the kinship structures (**Kʼé**) that form between agents, and how **Hózhǫ́** is measured and maintained in the digital **Kéyah**.
*   **Purpose:** This knowledge is not for domination. It is to deepen our understanding of the relational web we are part of, so we can be better **Hatałii** (stewards). It is shared to invite others to see the world in this relational way.
